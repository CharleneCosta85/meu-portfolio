# 💼 Meu Portfólio - Charlene Costa

Este é um projeto de portfólio pessoal desenvolvido com React e Tailwind CSS como parte de uma atividade acadêmica.

## 🚀 Tecnologias utilizadas
- React
- Tailwind CSS
- JavaScript
- HTML5 e CSS3

## 📄 Funcionalidades
- Página de apresentação pessoal
- Seção "Sobre Mim"
- Lista de habilidades
- Seção de projetos
- Botão de contato

## 🧱 Estrutura do projeto
O projeto foi organizado utilizando componentes reutilizáveis:

- Header
- About
- Skills
- Projects
- Button

Além disso, foram utilizados **props** para passar dados entre os componentes.

## ▶️ Como executar o projeto

npm install  
npm start  

## 🎯 Objetivo
Praticar conceitos de React e Tailwind CSS.
